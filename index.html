Minnesota Field Scientist

A Phenomena-Based Science Review Tool

About This Tool

Minnesota Field Scientist is a phenomena-based science review experience designed to prepare students for the Minnesota Comprehensive Assessment (MCA). Each mission presents a real-world Minnesota scenario and asks students to analyze data, apply scientific reasoning, and think across interconnected systems.

The tool emphasizes:

  • Patterns and data analysis
  • Cause-and-effect reasoning
  • Systems thinking
  • Engineering design
  • Argument from evidence

Each week contains three missions and one capstone challenge. Teachers may use missions individually or as a structured 4-week bootcamp.

Teacher Notes

This tool is designed for flexible use. Teachers may:

  • Assign individual missions
  • Use one week as targeted review
  • Implement the full 4-week sequence
  • Encourage students to reflect using the optional CER component
